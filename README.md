# 🤖 ARC Chatbot (Python Voice Assistant)

A simple Python-based AI chatbot with **text + voice interaction**. It allows users to chat in the terminal, and the bot responds using both **text output and speech**. Perfect for beginners exploring AI and voice assistants.

---

## 📷 ARC Chatbot

A lightweight chatbot that can:

* Respond to basic user queries
* Speak responses using text-to-speech
* Run directly in terminal

---

## 🚀 Features

* 💬 Chat in terminal (interactive mode)
* 🔊 Text-to-Speech (voice output)
* ⚡ Fast and lightweight
* 🧠 Predefined intelligent responses
* 💻 Beginner-friendly project
* 🪟 Works best on Windows (uses `sapi5`)

---

## 🛠 Tech Stack

* Python 3.x
* `pyttsx3` (Text-to-Speech library)

---

## 📦 Installation

### 1️⃣ Install required module:

```bash
pip install pyttsx3
```

### 2️⃣ Clone the project:

```bash
git clone https://github.com/codertheashish/ARC_Chatbot.git
```

### 3️⃣ Navigate to project folder:

```bash
cd ARC_Chatbot
```

---

## ▶️ How to Use

Run the script:

```bash
python chatbot.py
```

You will see:

```
ARC CHATBOT is running... Type 'exit' to quit.
```

### 💬 Example Interaction:

```
You: hello
Chatbot: Hello! How are you?

You: tell me a joke
Chatbot: Joke: Why do not programmers like nature? It has too many bugs.
```

---

## 📁 Output

✔ Text response in terminal
✔ Voice response through speakers

---

## 📌 Notes

* Works best on **Windows** (uses `sapi5` voice engine)
* Voice speed is adjustable (`rate` property in code)
* Supports basic conversational queries
* Type `exit` to stop the chatbot
